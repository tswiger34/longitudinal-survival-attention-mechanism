# Longitudinal Transformer for Survival Analysis

A generalization of the LTSA model as proposed in [Harnessing the Power of Longitudinal Medical Imaging for Eye Disease Prognosis Using Transformer-based Sequence Modeling](https://www.nature.com/articles/s41746-024-01207-4)

All credit to Gregory Holste, Mingquan Lin, Ruiwen Zhou, Fei Wang, Lei Liu, Qi Yan, Sarah H Van Tassel, Kyle Kovacs, Emily Y Chew, Zhiyong Lu, Zhangyang Wang, Yifan Peng for proposing this model and conducting the initial research required. You can find the repository for the initial implementation of the model and their research [here](https://github.com/bionlplab/longitudinal_transformer_for_survival_analysis).
